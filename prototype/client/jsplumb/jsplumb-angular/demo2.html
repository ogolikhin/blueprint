JsPlumb-Angular Demo

Using jsplumb draggable:

Apply directive 'jsplumb-draggable' to the elements you cant to be able to drag/drop. It will apply jsPlumbInstance.draggable() to the element.
You can pass options to the directive it will be used as a second argument in the 'jsPlumb.draggable(element, options)' call .
