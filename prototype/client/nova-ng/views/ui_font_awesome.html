Font Awesome version 4.4

Icon Sizes

Increase the icon size by using the fa-lg (33% increase), fa-2x, fa-3x, or fa-4x classes.

(default)

.fa-lg

.fa.fa-skyatlas.fa-2x

.fa.fa-skyatlas.fa-3x

.fa.fa-skyatlas.fa-4x

Rotated & Flipped Icons

To arbitrarily rotate and flip icons, use the fa-rotate-* and fa-flip-* classes.

.fa-rotate-90

.fa-rotate-180

.fa-rotate-270

.fa-flip-horizontal

.fa-flip-vertical

Bordered & Pulled Icons

Use fa-border and pull-right or pull-left for easy pull quotes or article icons.

Use a few of the new styles together, and you've got easy pull quotes or a great introductory article image. Or spinning icons for loading and refreshing content. Or fun big icons in multi-line buttons. You can combine all of them in any combination to get lots of new possibilities.

Animated Icons

Use the fa-spin class to get any icon to rotate, and use fa-pulse to have it rotate with 8 steps.
